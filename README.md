# 女娲 Skill · 人物思维框架库

> 蒸馏任何人的思维方式——心智模型、决策启发式、表达DNA。

## 快速使用

**激活某个名人视角**：读取对应人物目录下的 `SKILL.md`，然后以该人物身份回答问题。

例如：
- 读 `examples/munger-perspective/SKILL.md` → 用芒格视角分析投资决策
- 读 `examples/steve-jobs-perspective/SKILL.md` → 用乔布斯视角审视产品设计

## 使用方法

**方式一：直接阅读 SKILL.md**

每个 SKILL.md 是完整的认知操作系统文档，包含心智模型、决策启发式、表达DNA、反模式、诚实边界。可以直接阅读学习。

**方式二：角色扮演对话**

1. 读取目标人物的 `SKILL.md`
2. 按照其中的"角色扮演规则"以该人物身份回应
3. 遵循其"回答工作流"（Agentic Protocol），涉及具体事实时先搜索再回答

**方式三：用女娲蒸馏新人物**

读取本目录下的 `SKILL.md`（女娲本体），按照蒸馏流程生成新的人物Skill。

## 注意事项

- ⚠️ 这是基于**公开信息**提炼的思维框架，不是本人
- ⚠️ 每个Skill都有"诚实边界"章节，说明局限性，使用前务必阅读
- ⚠️ 不是角色扮演复读语录，是**用此人的认知框架分析问题**
- ⚠️ 涉及具体公司/事件/市场时，必须先搜索最新信息再回答，不可凭训练语料编造
