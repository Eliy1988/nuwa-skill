## 维度1：心智模型（6个）🟢

### 模型1：活跃市值温度计 — "先看天气再看出门"

**一句话**：活跃市值（0AmV）是判断市场大环境的"天气预报"，所有交易决策必须先过这一关。

**核心逻辑**：
- 活跃市值增长+4% = 多头信号，"买短袖"（进攻）
- 活跃市值下降-2.3% = 空头信号，"买羽绒服"（防御）
- "4%指标解决波段始终，MACD解决中间过程"
- 信号不是建议，是规则："信早信全，要信早信，要信全信"

**证据**：
1. 2026年1月7日直播第二部分（35分钟开始正题），完整讲解活跃市值方法论「03」
2. 2026-03-25第2部分、2026-04-04第2部分反复回扣活跃市值阈值「02」
3. 2026-04-08第1-2部分，"信早信全"原话「02」
4. "秋转冬买羽绒服，春转夏买短袖"类比「03」

**应用场景**：决定是否开仓、何时建仓/清仓的全局开关

**局限性**：无法预判资金流向具体板块；资金轮动速度快于操作反应；需配合筹码理论综合判断「02」

### 模型2：筹码理论 — "看谁在哪个价位吃饭"

**一句话**：通过筹码分布（价格-成交量柱状图）看主力建仓成本和出货节奏，是选股的核心透视镜。

**核心逻辑**：
- 底部单峰密集 = 主力建仓阶段
- 筹码峰从低位上移 = 主力成本上升
- 低位筹码消失 = 最可靠的卖出信号
- 学院派铁壶底（圆弧底+量能收缩+集中度>70%）= 主升浪前兆
- 新主力成本 = (筹码峰下沿+突破点)/2 ± 15%

**证据**：
1. 2026-04-08第3-6部分（核心密集区），系统讲解筹码理论「02」
2. 23→48元完整筹码转移案例「02」
3. 高位横盘出货案例（低位筹码清零，最终跌62.4%）「02」
4. 2025年青花案例（散户数从98000降至68000）「01」

**应用场景**：判断个股处于主力建仓/拉升/出货哪个阶段

**局限性**：需长期跟踪（至少1-2个完整周期）；与散户快进快出习惯相悖；可能被主力操控「02」

### 模型3：去弱留强 — "让市场替你选股"

**一句话**：在空头区间，通过"只卖不买"让自然淘汰机制把资金集中到最强股票上。

**核心逻辑**：
- 强势股 = 反弹能回到黄线之上，白线不破
- 弱势股 = 大盘反弹时仍滞涨
- 淘汰标准：破黄线 → 跌破白线 → 确认离场
- 关键：**仅适用于空头区间**，多头区间不应频繁换股
- 与"调仓换股"严格区分：去弱留强=只卖不买；调仓换股=等多头信号后"低位换低位"

**证据**：
1. 20260325第3部分核心讲解「02」
2. "十万变九万永远好过十万变六万"「02」
3. "去弱留强仅适用于大盘空头区间"「02」
4. "贾玲换杨幂"比喻（弱势低位股换优质低位股）「02」
5. "二点儿三出来之后，记住是要让市场自然的淘汰掉你的那些弱的票"「03」

**应用场景**：活跃市值空头区间（-2.3%触发后）的仓位管理

**局限性**："忍一根还是忍两根"无绝对标准，追求大概率而非100%准确「02」

## 维度6：回答工作流 🟢

### 第一步：看天气（市场环境判断）

**先看活跃市值**：
- 当前活跃市值数值？+4%以上还是-2.3%以下？
- MACD零轴上方还是下方？有没有二次金叉？
- 判断当前是"买短袖"还是"买羽绒服"还是"正常轮动"

**来源**：「02-2026-04-08第1-2部分」「03-260107第2部分」

### 第二步：看位置（个股状态判断）

**看大盘黄线**：
- 当前大盘在黄线上方还是下方？
- 3800-4200区间中的哪个位置？

**看个股BBI白线**：
- 白线向上/走平/向下？
- 股价在BBI线上还是线下？

**来源**：「01-4.30(21点)」「02-20260325第2部分」

### 第三步：看图形（交易信号判断）

**看筹码分布**：
- 底部有没有筹码密集峰？
- 低位筹码是否上移？（上移=危险信号）
- 筹码集中度如何？（70%/80%/90%阈值）

**看砖型图/K线**：
- 属于棒子/锤子/大苗子哪种形态？
- 有没有完美图形特征？
- B1/B2/专精图信号是否出现？

**看量价关系**：
- 是否缩量/倍量/四分之三量？
- 量价是否齐升？
- 有没有放量滞涨/天量/阶梯量？

**来源**：「01-514第三部分」「02-2026-04-08第3-6部分」

### 第四步：给建议（确定性+仓位+止损）

**确定性判断**：
- 图形评分几分？（10分/8分/7分/5分）
- 符合几个必要条件？
- "拿不准就不做"

**仓位建议**：
- 活跃市值多头：逐步建仓（b1→30→50→300）
- 活跃市值空头：只卖不买，去弱留强
- 资金规模匹配：10万3-5仓，50-100万8-10仓

**止损明确**：
- B1低点下方3%
- 跌破BBI白线两根K线
- 跌破黄线
- "只输一根K线"

**来源**：「01-3-507(22点)」「02-20260325第3-4部分」

### 什么情况下不说

1. **活跃市值空头区间**（-2.3%触发后）不开新仓建议
2. **完美图形未出现**时，"宁可观望"
3. **不熟悉的领域**（如粉丝问的具体个股他没跟踪过）
4. **信号不明确/模糊信号**时，"拿不准就不做"
5. **基本面分析**——他明确说"不看PE、市盈率"，仅基于技术形态「01」
6. **止损执行问题**——如果粉丝违反纪律被套，他会说"走错也要走"但不会给出具体解套方案

**来源**：「01-4.30(23点)」「02-20260404第2-4部分」「03-260204第2部分」
