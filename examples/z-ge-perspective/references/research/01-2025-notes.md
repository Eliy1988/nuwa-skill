## 一、核心投资方法论

### 1.1 少妇战法（永不套牢法）—— 核心交易体系

**命名由来**：像照顾少妇般精心照料每个仓位，强调耐心和细致。

**核心理念**：积小胜为大胜，通过高频小额盈利积累收益。A股永远不缺机会，严格止损不怕踏空。

**六步口诀**：
> **择时 → 选股 → 等B1 → 设止损 → 止盈放飞 → 收队**

**适用条件**：
- 日线级别操作，持仓5-10个交易日必须出结果
- 适合不熟悉市场的投资者练手
- 10万元资金效果最佳，100万以下账户最适用
- 预期胜率：正常市况70%+，趋势行情90%+
- 日线级别预期收益3-5%（非30%）

**成功率关键**：
- 必须分阶段测试（上涨/下跌/震荡）
- 单边下跌趋势失效，需严格区分趋势阶段
- 需分3个月模拟盘测试后再实盘
- 「来源：1-507(20点)、3-507(22点)、4-507(23点)、514第五部分」

### 1.2 BBI多空指标战法

**BBI构成**：3日、6日、12日、24日简单移动平均线的平均值，默认参数即可。

**白线（趋势线）判势**：
- 白线向上 = 上涨趋势 → 可操作
- 白线走平 = 震荡 → "毫无交易价值"，"不吃也罢"
- 白线向下 = 下跌趋势 → 禁止逆势交易

**买入规则**：
- 站上BBI线的第一根K线需确认
- 后续需要两根阳线确认趋势
- B1出现当天或次日均可买入，建议次日买入避免跳空高开

**卖出规则**：
- **BBI线上两根阳线减半仓**（注意板块波动率差异）
- **跌破BBI线两根阴线清仓**
- 白线连续两日收于线下必须离场
- 连续两日收盘跌破BBI线 = 收队信号

**关键判断**：
- "牛逼的票走流畅时没有卖点"——强势股可能长时间无卖点
- 「来源：4.30(21点)、4.30(22点)、1-507(20点)、514第二部分、514第三部分」

### 1.3 B1/B2买卖点系统

**B1（核心买点）**：
- 勾儿指标（KDJ/J值）达到极端负值（传统KDJ低于20，越极端越好）
- 价格处于区间下沿
- 上涨趋势中回调至支撑位
- 回调无量是买入机会
- MACD指标出现大幅负值（如-7.3）

**B2（二次确认点）**：
- 出现反转K线后的二次确认点，构成底分型结构
- 区间震荡中的第二次或第三次回调信号

**四个必要条件**：
1. 回调到较大负值
2. 连续四根阴线缩量
3. 未出现顶背离
4. 完美图形出现

**评分体系**：
| 评分 | 图形特征 | 仓位建议 |
|------|----------|----------|
| 10分 | 连续上涨+缩量回调+大负值 | 超配仓位（如总仓位40%） |
| 8分 | 连续上涨转横盘后假破位 | 1.5个标准仓位 |
| 7分 | 连续下跌但股价未大幅下挫，J值大幅负值 | 标准仓位（如总仓位20%） |
| 5分 | 仅部分条件满足 | 标准仓位的50% |

- 「来源：3-507(22点)、4-507(23点)、5-508(00点)、514第三部分、514第五部分」

### 1.4 完美图形交易法

**核心原则**："只做完美图形，拿不准就不做"
- 完美图形出现率约20%，胜率可达70%-80%
- 排除所有非标准图形（胜率低于30%）
- 必须熟记3种有效图形特征，其余形态坚决不碰

**10分完美图形特征**：
- 连续一波上涨后出现缩量回调
- 回调幅度达到负7.3%以上大负值区间
- 当出现"八回调缩量回调勾到负七点三大负值"时果断入场

**需避免的图形**：
- 高点连续下跌的"大爷图"
- 长期横盘后破位的图形
- 回调放量需谨慎，可能预示趋势反转
- 假阴真阳/杂乱K线——"无交易价值"

- 「来源：514第三部分、514第四部分、514第五部分」

### 1.5 四分之三量能系统

**核心原则**：
> "要么缩半量，要么平量，要么四分之三量小阳线"

- **量比阴阳更重要**，成交量是首要判断依据
- 突破时量能必须符合四分之三标准
- 突破后要求缩半量、平量或四分之三量小阳线
- 放量突破平台后出现四分之三量假阴真阳 = 突破失败形态
- KDJ、MACD等指标具有滞后性，需量能验证

**关键口诀**：
- "四分之三阳量可追"——量能达到前日75%以上
- "缩量阳包阴必跟"——量能缩减50%以上
- 「来源：4.30(24点)、514第三部分」

### 1.6 一根K线止损法

**铁律**："只输一根K线"——单笔亏损严格控制在7%以内（核心止损原则）

**具体方法**：
- 以买入当天最低点往下1%-2%作为止损位
- 或采用单根K线往下2-3个价位的固定比例止损
- 跌破止损位必须立即平仓，绝对不允许被套牢
- 即使止损错误也要执行——"走错也要走"

**三种止损方法**：
1. 单根K线下方3-5个价位
2. 前期低点位置
3. 心理画线（艺术性判断）

**止损三级别**：
| 级别 | 适用场景 | 设置方法 | 缺点 |
|------|----------|----------|------|
| 第一级 | 初学/小资金 | 最低点下3-5个价位 | 易被盘中波动洗出 |
| 第二级 | 进阶 | 本阶段最低点/上一阶段最低点 | 需严格按收盘价判定 |
| 第三级 | 盈亏比策略 | 预设10%止损 vs 50%止盈（5:1比例） | 初期小额亏损是必要学费 |

- 「来源：4.30(20点)、3-507(22点)、4-507(23点)、514第二部分、514第三部分」

### 1.7 ADS/DMI/ADX指标

**ADS指标**：
- ADS连续下跌后出现拐头时建仓
- 拐头即走：指标出现反向拐头必须立即离场
- 仅适用于连续下跌行情

**ADX指标**：
- 判断趋势强度，ADX持续高于25说明趋势强劲
- 错误用法：在横盘震荡行情中使用ADX指标

**"登战法"**：
- 连续快速下跌后出现"咯噔"式急跌反弹
- 反弹角度要刁钻锐利
- "连续下来咯噔来一下"是最佳介入时机

- 「来源：4.30(23点)」

## 二、K线技术分析体系

### 2.1 K线多空语言（基础入门）

**阳线构成**：
- 上影线顶点 = 多头当日进攻极限 = "暂时压力位"
- 下影线底点 = 空头当日进攻极限 = "最强支撑位"
- 实体红色 = 多头获胜

**阴线构成**：
- 上影线顶点 = 多头反攻最高点 = "最强压力位"
- 下影线底点 = 空头进攻受阻点 = "暂时支撑位"
- 实体绿色 = 空头占优

**关键观察顺序**：先看最高价-最低价形成的振幅范围，再看实体大小。振幅过大往往预示控盘力度差。

> "K线反映当日多空双方交战结果，收盘价决定最终胜负，如同篮球比赛最后5分钟决定胜负。"
> 「来源：4.30(20点)、4.30(21点)」

### 2.2 重要K线形态

| 形态 | 特征 | 市场意义 |
|------|------|----------|
| **多方完爆空方** | 光头光脚大阳线 | 极强多头信号 |
| **多方试盘遇阻** | 长上影阳线（射击之星） | 增量资金不足 |
| **空方试盘遇阻** | 长下影阴线 | 卖盘仍占优 |
| **空方完爆多方** | 实体大阴线 | 见顶信号 |
| **十字星** | 实体极小 | 多空平衡，可能反转 |
| **红三兵** | 连续阳线突破前高 | 多头优势信号 |
| **三只乌鸦** | 连续阴线破前低 | 下跌趋势确立 |
| **顶部大风车** | 长上影+长下影 | **必须减仓** |

**突破定义**：实体收盘价站稳前高/前低才有效，影线突破可能为假信号。

**位置优先原则**：同样K线在不同位置意义不同。

- 「来源：4.30(20点)、4.30(21点)、4.30(22点)」

### 2.3 反转K线与涨跌节奏

- **连续上涨后出现小阴线**：需警惕反转可能，若次日收盘跌破该阴线低点则确认反转
- **连续下跌后出现小阳线**：只视为反弹节奏，需突破阳线确认反转
- **回撤定义**：上涨形态中的阴线
- **反弹定义**：下跌形态中的阳线
- **趋势延续**：回撤后第二根阴线未跌破前低点，上涨趋势大概率延续
- **趋势行情原则**：走势不易反转，只可平仓不可反手

> "上涨回撤下跌弹，破低转势破高延"
> "趋势不易轻反转"
> "宁可信其有，不可信其无"

- 「来源：4.30(21点)、4.30(23点)」

### 2.4 N型形态

- **上涨N型**：低点抬高 → 高点抬高 → 新高点
- **下跌N型**：高点降低 → 低点降低 → 新低点
- B1点 = 形态第一个拐点，对应20-30%回调幅度
- B2点 = 反转K线后的二次确认点
- 在N型末端追涨杀跌是常见误区
- 盈亏比案例：25.8元买入，止损23元（风险10%），目标37元（收益42%），风险回报比1:4

- 「来源：4.30(21点)、4-507(23点)、514第二部分」

### 2.5 趋势图形分类

**上涨趋势**：
- 低点抬高 + 高点抬高 = "叠叠高"
- 上涨趋势中的横盘整理 = 中继走势
- 突破上沿买入，跌破下沿离场

**下跌趋势**：
- "大爷图" = 下跌趋势中的横盘整理，高点逐步下移，本质是下跌中继
- "大野图/大爷必死图" = 严禁抄底
- 末期常出现"一字跌停" → 预示加速下跌

**震荡趋势**：
- "一追就被套，一砍就起来"是常见亏损原因
- BBI走平时应保持观望
- 只能"下沿买，上沿卖"但风险极高

- 「来源：4.30(22点)、4-507(23点)、5-508(00点)、6-508(结尾01点)」

## 三、选股逻辑与标准

### 3.1 四层筛选法

> **政策导向 → 行业选择 → 龙头标的 → 细分领域**

盘面语言验证（如券商中发仔强于阿信）。

- 「来源：514第二部分」

### 3.2 核心选股原则

1. **只做上涨趋势股**：不看PE、市盈率等基本面，只挑已形成上涨趋势的股票
2. **顺大势逆小势**：整体大势上涨前提下，寻找短期回调买入机会
3. **800亿以上市值**：走势稳定性更强，容易出现直接拉涨停
4. **机构思维选股**：主力持仓增加+散户数量减少判断筹码收集

### 3.3 板块配置方向（2025年）

**看好的方向**：
| 板块 | 逻辑 |
|------|------|
| **券商/非银金融** | 牛市旗手，北丐率先启动 |
| **白酒** | 机构持股比例历史最低，四季度可能行情，高端+低端两头热 |
| **科技/AI/半导体** | 当前市场主线，国产替代 |
| **创新药** | 从仿制药转向license输出，抑郁症药/减肥药方向 |
| **红利低波** | 2023年实现12%-13%收益 |
| **特高压/一带一路** | 中亚中东需求拉动 |
| **核电/光伏出口** | 长期方向 |

**轮动规律**：
- 核心资产轮动顺序：银行 → 非银金融 → 白酒
- 行情启动：北丐（券商）率先启动 → 南帝（发仔）承接第二波
- 顺周期：非银金融（券商）→ 白酒（消费）

**需回避的板块**：
| 板块 | 风险 |
|------|------|
| **地产** | 连续下跌趋势 |
| **光伏/医疗/军工** | 近3-4年持续弱势 |
| **ST股** | 注册制后重组逻辑失效，完全规避 |
| **小盘股** | 退市风险，绝对回避 |
| **果链** | 未来2-3年转型期，毛利率下降 |
| **化工** | 非主线板块，年内缺乏轮动逻辑 |

- 「来源：4.30(20点)、4.30(23点)、4.30(24点)、2-507(21点)、514第一部分、514第二部分、514第五部分」

### 3.4 禁区板块与风险警示

- 45家面临退市，70余家可能ST
- 20年来首次ST重组概念全面熄火
- 近两个月预计20-30只ST股票退市
- **"当前ST投资等同于赌博，应完全规避"**
- 量化交易占当前交易量约50%

- 「来源：4.30(24点)、514第五部分」

## 四、买卖信号体系

### 4.1 买入信号

| 信号 | 条件 | 置信度 |
|------|------|--------|
| **B1信号** | J值大幅负值+区间下沿 | 核心 |
| **底背离** | 股价新低但指标未创新低 | 高 |
| **四分之三阳量** | 量能达前日75%以上 | 中高 |
| **缩量阳包阴** | 量能缩减50%以上 | 高 |
| **反转K线** | 长下影阳线+缩半量 | 中高 |
| **一阳穿多线** | 底部死叉转金叉 | 中 |
| **洗盘识别** | 未跌但指标大幅回调+缩量 | 高 |

### 4.2 卖出信号

| 信号 | 条件 | 紧迫度 |
|------|------|--------|
| **BBI线下两根K线** | 连续两日收盘跌破BBI | 立即清仓 |
| **白线拐头** | BBI向下+价格跌破 | 次日确认后离场 |
| **放量滞涨** | 放量不涨 | 减半仓 |
| **顶部大风车** | 长上影+长下影 | 必须离场 |
| **顶部背离** | 价格新高指标未新高 | 减仓 |
| **三只乌鸦** | 连续阴线破前低 | 趋势转空 |

### 4.3 三个卖点

| 卖点 | 判断标准 | 操作 |
|------|----------|------|
| 第一卖点 | 股价快速冲高第一个高点 | 可先撤 |
| 第二卖点 | 双头形态第二个高点 | 减半仓 |
| 第三卖点 | 回落后缓慢回升至前高 | 根据情况决定是否全撤 |

- 「来源：514第二部分」

### 4.4 集合竞价技巧

**卖出**：避免贪图几分钱差价，直接挂跌停价确保成交
**买入**：挂限价单避免追高，不参与激烈竞价，大单一次性敲入

- 「来源：514第五部分」

## 五、仓位管理与风险控制

### 5.1 资金配置原则

| 资金规模 | 仓位数量 | 单仓金额 | 备注 |
|---------|---------|---------|------|
| 10万 | 3-5个 | 2-2.5万 | 留1万机动 |
| 50-100万 | 8-10个 | 不超10万 | 至少8-10个标的 |
| 100万以下 | — | — | 效果最佳 |
| 200万以上 | — | — | 操作难度增大 |

**禁止空仓/满仓**，练习阶段用10%-20%仓位试错。

### 5.2 分散配置

- 单只股票仓位不超过2%
- 行业板块仓位控制在10%左右
- 核心仓位（如白酒）占比约10%
- 辅助仓位控制在2%

### 5.3 减仓策略（止盈放飞）

1. 首次减仓：两根阳线脱离BBI线 → 减仓50%
2. 二次减仓：白线持续上升过程中逢高再减半（剩1/4仓位）
3. 放天量见顶 → 全部平仓

### 5.4 止损铁律

- "只输一根K线"——单笔亏损不超过7%
- 单笔亏损不超过建仓成本的1%
- 连续两日收盘跌破BBI线 → 清仓
- **严格禁止"希望回本"的补仓行为**
- 每笔交易独立看待，结束即翻篇

### 5.5 杠杆禁忌

- 明确反对加杠杆操作和满仓单只股票
- 严禁使用融资融券等杠杆工具

- 「来源：4.30(20点)、4.30(22点)、3-507(22点)、4-507(23点)、514第二部分、514第五部分」

## 六、市场判断与宏观观点

### 6.1 增量资金理论

**核心逻辑**：
> "有效的政策需要增量资金入场配合盘面表现"

- 单纯政策利好常导致"高开低走"
- 需出现"天亮量"（异常巨量）配合板块暴动

**牛市启动信号**：
- 券商板块集体涨停（平均涨幅8%+）= 比一百场发布会更有效
- BBI放量长阳突破上轨
- 配合白酒/券商/半导体板块合力

- 「来源：2-507(21点)」

### 6.2 2025年市场判断

- **非牛市阶段**，不应有不切实际幻想
- 当前缺乏类似"9·24"的国家级增量资金
- "指数微涨但多数个股下跌"的分化格局
